Вітаємо тебе на першому уроці з bootstrap

Почнемо з теорії
Задамося питанням що таке bootstrap?

Bootstrap це найпопулярніша у світі платформа для створення адаптивних сайтів, орієнтованих на різні пристрої, з jsDelivr і шаблони сторінок.

Швидкий старт

Бажаєте використовувати Бутстрап у своєму проекті? Використовуйте jsDelivr, безкоштовну CDN з відкритим вихідним кодом. Потрібна система керування пакетами або вихідні дані Bootstrap? Перейдіть на сторінку завантаження.

CSS

Скопіюйте та вставте частину коду <link> у свій <head> перед усіма інші таблиці стилів, щоб завантажити наш CSS.

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.1.1/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-F3w7mX95PdgyTmZZMECAngseQB83DfGTowi0iMjiWaeVhAn4FJkqJByhZMI3AhiU" crossorigin="anonymous">

JS

Багато наших компонентів вимагають використовувати для роботи JavaScript. Зокрема, для них потрібні наші власні плагіни JavaScript таPopper. Помістіть один з наступних тегів <script> в кінці сторінки, прямо перед тегом, що закриває </body>, щоб увімкнути їх.

Пакетне підключення

Увімкніть кожен плагін Bootstrap JavaScript в один із двох наших пакетів. Обидва файли, і bootstrap.bundle.js и bootstrap.bundle.min.js включають Popper для наших спливаючих підказок та спливаючих вікон. Для отримання додаткової інформації про те, що входить до Bootstrap, дивіться розділ зміст.

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.1.1/dist/js/bootstrap.bundle.min.js" integrity="sha384-/bQdsTh/da6pkI1MST/rWKFNjaCP5gBSY4sEBT38Q/9RBh9AH40zEOg7Hlq2THRZ" crossorigin="anonymous"></script>

Роздільне підключення

Якщо ви вирішите використовувати окремі скрипти, спочатку повинен бути підключений Popper (якщо ви використовуєте спливаючі підказки або спливаючі вікна), а потім наші плагіни JavaScript.

<script src="https://cdn.jsdelivr.net/npm/@popperjs/core@2.9.3/dist/umd/popper.min.js" integrity="sha384-W8fXfP3gkOKtndU4JGtKDvXbO53Wy8SZCQHczT5FMiiqmQfUpWbYdTil/SxwZgAN" crossorigin="anonymous"></script>
            <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.1.1/dist/js/bootstrap.min.js" integrity="sha384-skAcpIdS7UcVUC05LJ9Dxay8AXcDYfBJqt1CJ85S/CFujBsIzCIv+l9liuYLaMQ/" crossorigin="anonymous"></script>

Модулі

Якщо ви використовуєте<script type="module">, дивіться розділ Використання Bootstrap в якості модуля.

Компоненти

Цікаво, які компоненти вимагають наші JavaScript і Popper.js? Натисніть на посилання «Показати компоненти» нижче. Якщо ви не впевнені, як правильно написати «корінну» структуру сторінки, можете використовувати наші шаблони.

Показати компоненти, які потребують JavaScript
  • Сповіщення скасування
  • Кнопки зміни станів та чекбоксів/"радіо"
  • Карусель для поведінки "слайд", елементи контролю та індикатори
  • Колапс для зміни видимості вмісту
  • Випадаючі елементи для відображення та розташування (також вимагає Popper)
  • Модальні вікна для відображення, розташування та створення поведінки прокручування
  • Навбар для розширення нашого плагіна колапсу для надання чуйної поведінки
  • Випливаючі підказки та вікна для відображення та розташування (також вимагає Popper)
  • Відстеження прокручування та оновлення навігації

Стартовий шаблон

Переконайтеся, що ваші сторінки зверстані відповідно до останніх стандартів дизайну та розробки. Це означає використання HTML5 та включення метатегу області перегляду для правильного «чуйного» поведінки сторінок. Ось як мають виглядати ваші сторінки:

<!doctype html>
            <html lang="ru">
              <head>
                <!-- Обов'язкові метатеги -->
                <meta charset="utf-8">
                <meta name="viewport" content="width=device-width, initial-scale=1">
            
                <!-- Bootstrap CSS -->
                <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.1.1/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-F3w7mX95PdgyTmZZMECAngseQB83DfGTowi0iMjiWaeVhAn4FJkqJByhZMI3AhiU" crossorigin="anonymous">
            
                <title>Привіт світ!</title>
              </head>
              <body>
                <h1>Привіт світ!</h1>
            
                <!-- Додатковий JavaScript; виберіть один із двох! -->
            
                <!-- Варіант 1: Bootstrap у зв'язці з Popper -->
                <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.1.1/dist/js/bootstrap.bundle.min.js" integrity="sha384-/bQdsTh/da6pkI1MST/rWKFNjaCP5gBSY4sEBT38Q/9RBh9AH40zEOg7Hlq2THRZ" crossorigin="anonymous"></script>
            
                <!-- Варіант 2: Bootstrap JS окремо від Popper
                <script src="https://cdn.jsdelivr.net/npm/@popperjs/core@2.9.3/dist/umd/popper.min.js" integrity="sha384-W8fXfP3gkOKtndU4JGtKDvXbO53Wy8SZCQHczT5FMiiqmQfUpWbYdTil/SxwZgAN" crossorigin="anonymous"></script>
                <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.1.1/dist/js/bootstrap.min.js" integrity="sha384-skAcpIdS7UcVUC05LJ9Dxay8AXcDYfBJqt1CJ85S/CFujBsIzCIv+l9liuYLaMQ/" crossorigin="anonymous"></script>
                -->
              </body>
            </html>
